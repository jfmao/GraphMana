# graphmana neo4j-start

## Synopsis

```
Usage: graphmana neo4j-start [OPTIONS]
```

## Help Output

```
Usage: graphmana neo4j-start [OPTIONS]

  Start Neo4j in user space.

Options:
  --neo4j-home DIRECTORY  Neo4j installation directory.  [required]
  --data-dir DIRECTORY    Override data directory.
  --wait / --no-wait      Wait until Neo4j is ready.
  --timeout INTEGER       Max seconds to wait.
  --verbose / --quiet     Verbose logging.
  --help                  Show this message and exit.
```
