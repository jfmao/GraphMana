# graphmana snapshot create

## Synopsis

```
Usage: graphmana snapshot create [OPTIONS]
```

## Help Output

```
Usage: graphmana snapshot create [OPTIONS]

  Create a database snapshot via neo4j-admin dump.

Options:
  --name TEXT               Snapshot name (unique).  [required]
  --neo4j-home DIRECTORY    Neo4j installation directory.  [required]
  --database TEXT           Neo4j database name.
  --snapshot-dir DIRECTORY  Directory to store snapshots.
  --help                    Show this message and exit.
```
