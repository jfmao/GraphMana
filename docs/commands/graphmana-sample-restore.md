# graphmana sample restore

## Synopsis

```
Usage: graphmana sample restore [OPTIONS]
```

## Help Output

```
Usage: graphmana sample restore [OPTIONS]

  Restore soft-deleted samples (clear excluded flag).

Options:
  --sample-ids TEXT      Sample IDs to restore (repeatable).
  --sample-list FILE     File with sample IDs (one per line).
  --neo4j-uri TEXT       Neo4j Bolt URI.
  --neo4j-user TEXT      Neo4j username.
  --neo4j-password TEXT  Neo4j password.
  --database TEXT        Neo4j database name.
  --help                 Show this message and exit.
```
