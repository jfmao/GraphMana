# graphmana init

## Synopsis

```
Usage: graphmana init [OPTIONS]
```

## Help Output

```
Usage: graphmana init [OPTIONS]

  Initialize a new GraphMana project directory with standard structure.

Options:
  --project-dir PATH     Directory for the new GraphMana project.  [required]
  --install-neo4j        Download and install Neo4j into the project
                         directory.
  --neo4j-password TEXT  Initial Neo4j password. Default: graphmana.
  --database TEXT        Neo4j database name.
  --help                 Show this message and exit.
```
