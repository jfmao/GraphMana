# graphmana cohort

## Synopsis

```
Usage: graphmana cohort [OPTIONS] COMMAND [ARGS]...
```

## Help Output

```
Usage: graphmana cohort [OPTIONS] COMMAND [ARGS]...

  Manage named cohort definitions.

Options:
  --help  Show this message and exit.

Commands:
  count     Count samples matching a named cohort.
  define    Define or update a named cohort.
  delete    Delete a named cohort.
  list      List all cohort definitions.
  show      Show details of a named cohort.
  validate  Validate a Cypher query for cohort use.
```
